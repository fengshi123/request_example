# request_example
本实例我们主要使用Express解析 Post 请求的 4 种 Content-Type 数据类型： 

 ● application/x-www-form-urlencoded
 
 ● multipart/form-data
 
 ● application/json
 
 ● text/xml
 
## 构建步骤

需要 Node.js 7+ （作者使用的版本为：node.js 8.2.1 npm 5.3.0）

安装插件：npm install

运行示例：npm run dev

浏览器打开：http://localhost:3000
